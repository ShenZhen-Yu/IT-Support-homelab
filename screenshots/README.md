# Screenshot Guide

## Purpose
This folder contains screenshots that document the setup, troubleshooting, and administration steps completed in the lab.

## Suggested Screenshots
### Networking
- Server IP settings
- Client IP settings
- Failed ping test
- Firewall setting change
- Successful ping test

### Server
- System information
- Server Manager
- Installed roles/features later on

### Client
- System information
- Network adapter settings
- Domain join screen later on

## Why Screenshots Matter
Screenshots provide proof of configuration, troubleshooting, and lab progress. They also make the GitHub project easier to understand.
